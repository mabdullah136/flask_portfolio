# 🌟 Flask Portfolio  

A **simple portfolio website** built with **Flask** and **MySQL** as the database.  

## 📌 Prerequisites  
Ensure you have the following installed:  
- **Python 3.x**  
- **MySQL**  
- **pip** (Python package manager)  
- **virtualenv** (Optional but recommended)
